# 8-bit-ALU-VLSI-project
## Project Description
This Project implements an 8-bit Arithmetic Logic Unit (ALU) using Verilog HDL

## Operations Performed 
- Addition
- Subtraction
- AND
- OR
- XOR
- NOT

## Tools Used
- Verilog HDL
- EDA Playground

## Simulation
The ALU was successfully simulated and verified using waveform outputs
